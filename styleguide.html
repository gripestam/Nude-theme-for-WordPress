Grid

We use Pure CSS (purecss.io/grids/) responsive grid system with some extra settings for visually boxing stuff. Included in _grid.scss.

The grid class is pure-g and unit classes is pure-u-*.

Pure ships with both a 5ths and 24ths based grid. See http://purecss.io/grids/#grids-units-sizes for more info.

Example
<div class="pure-g">
  <div class="pure-u-1 pure-u-md-1-3"> ... </div>
  <div class="pure-u-1 pure-u-md-1-3"> ... </div>
  <div class="pure-u-1 pure-u-md-1-3"> ... </div>
</div>
Standard breakpoints
Key CSS Media Query Applies Classname
None None Always .pure-u-*
sm @media screen and (min-width: 35.5em) ≥ 568px .pure-u-sm-*
md @media screen and (min-width: 48em) ≥ 768px .pure-u-md-*
lg @media screen and (min-width: 64em) ≥ 1024px .pure-u-lg-*
xl @media screen and (min-width: 80em) ≥ 1280px .pure-u-xl-*
